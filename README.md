# Sunny Antiques

Practice build before Doche.app — antique resale platform with AI-powered valuation.

## Stack
- **Frontend:** Next.js 14 (App Router)
- **Database:** Supabase (Postgres + Auth + Storage)
- **Image Storage:** Cloudinary
- **AI Valuation:** OpenAI Vision (GPT-4o)
- **Deployment:** Vercel

## Core Features
1. Item listing with photo upload
2. AI-powered valuation via image analysis
3. Search + filter by category, era, price range
4. Admin dashboard for inventory management
5. Buyer contact / inquiry form
